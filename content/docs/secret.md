# Секретное разделение

Разделение секрета по схеме Шамира (Shamir's Secret Sharing).

## Как это работает

Секрет разбивается на несколько частей (shares). Для восстановления нужна только часть из них (порог).

## Параметры

- **Порог (threshold)** — минимальное количество частей для восстановления
- **Части (shares)** — общее количество частей, на которые делится секрет

## Пример

При пороге 2 и 3 частях: секрет можно восстановить, имея любые 2 из 3 частей.
