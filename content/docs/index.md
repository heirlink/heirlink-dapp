# Heirlink — Документация

Связь с наследником — обеспечение сохранности цифрового капитала.

## Возможности

- **Распределение наследства** — создание Merkle tree для распределения токенов и ETH между наследниками
- **Секретное разделение** — разделение секрета по схеме Шамира для безопасной передачи

## Начало работы

1. Подключите кошелёк (MetaMask, Rainbow и др.)
2. Выберите нужный раздел в навигации слева
3. Следуйте инструкциям на странице
